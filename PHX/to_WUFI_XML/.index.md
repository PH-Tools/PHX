# PHX/to_WUFI_XML/ — write a PHX model to WUFI XML

Serializes the `../model/` graph to WUFI-Passive XML.

## Modules

- `xml_builder.py` — walks the PHX model and builds the XML tree.
- `xml_converter.py` — value/type conversion for XML output.
- `xml_schemas.py`, `xml_writables.py` — **PascalCase intentional** — mirror the WUFI XML / C# structure; do not rename to snake_case.
- `xml_txt_to_file.py` — write the assembled XML to disk.
- `_bug_fixes.py` — targeted WUFI-compatibility workarounds.

## Notes
- WUFI XML schema reference: `../../docs/reference/wufi-xml-schema.md`.
- Tests: `../../tests/test_to_WUFI_xml/` with parametrized `to_xml_reference_cases` (HBJSON in → expected XML out).

# PHX/from_WUFI_XML/ — build a PHX model from WUFI XML

Reads a WUFI-Passive XML file into the `../model/` graph using **pydantic v2** schemas.

## Modules

- `read_WUFI_XML_file.py` — load/parse the XML.
- `wufi_file_schema.py`, `wufi_file_types.py` — pydantic models mirroring the WUFI XML structure.
- `phx_schemas.py` — schema definitions bridging WUFI XML → PHX.
- `phx_converter.py` — populate the PHX model from the parsed schema.

## Notes
- The WUFI XML schema itself is documented in `../../docs/reference/wufi-xml-schema.md`.
- Tests: `../../tests/` (`from_WUFI/` reference files).
